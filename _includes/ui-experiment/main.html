An Experiment

This is a javascript/css experiment (no jQuery) on how to transform a section while keeping it clickable as well as flickable, while having a custom css cursor on the page that changes to the default in below 768px (for mobiles)

NEW: Added javascript class generator :D

This method works on IE11 and above. I can't be bothered to support IE :D
